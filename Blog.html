Question-1: What are the different ways to select an element in the DOM?

Ways to select an element in the DOM is:

  • getElementById
  • getElementsByClassName
  • getElementsByTagName
  • querySelector
  • querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: It is used to change or read the entire HTML code in an HTML element.
  • innerText: It only returns or sets text and ignores HTML tags.
  • textContent: This is similar to innerText, but it also returns the text of elements with CSS display: none;

Question-3: What is event delegation in the DOM?

  • Event delegation in the DOM is a JavaScript technique that uses the "event bubbling" process to attach an event listener to a parent element,
    allowing it to handle events triggered on its child elements, rather than adding a separate listener to each child element.

Question-4: What is event bubbling in the DOM?

  • Event bubbling is a process of DOM event propagation where when an event occurs on a child element, the event is first handled on that
    child element and then gradually propagates to its parent elements until it reaches the document.

Question-5: How do you create, add, and remove elements using JavaScript?

  • document.createElement("tagName") - A new element can be created using this command.
  • appendChild() – Add as the last child of the parent
  • remove() – Deletes an element directly
